Displays current GitHub service status on the M5Stack Tab5. Status refreshes every 5 minutes or when the screen is touched. Uses the [ArduinoJson](https://github.com/bblanchon/ArduinoJson) library.

Add your WiFi credentials to the source before compiling and uploading to the device:

```
// WiFi credentials
const char* ssid = "WIFISSID";
const char* password = "WIFIPASSWORD";
```
